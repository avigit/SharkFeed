Features

Launch Screen

- Swipe right to navigate

Shark Feed Screen

- Infinite scrolling
- Pull to refresh
- Lighbox to show high quality photo
- Custom animation for the transition to lightbox

Lightbox Screen

- Download high quality photo in photos
- Photo information displayed
- Zooming
- Allow screen rotation for better visibility
- Open photo in flickr from the lightbox page
- Share photo from lightbox

Backend Features

- Automatic serialization and deserialization for json response
- Cache photo for future use
- Download low quality photo first
- Handle network status changes
- Custom pull to refresh for better performance
- Test cases for testing automatic json serialization and deserialization
